### D06172604 Map Type v1

`Map(key_type, value_type, value_contains_null=True)` is included in the v1 schema type surface. Structure should
generate Spark `MapType` declarations, validate map key and value types where Spark metadata allows it, and keep map
keys non-null because Spark map keys cannot be null.

This decision covers schema declaration, schema generation, and runtime validation only. Spark higher-order map
transformations remain a v2 expression feature.
