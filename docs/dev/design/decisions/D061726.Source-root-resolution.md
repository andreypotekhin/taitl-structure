### Source Root Resolution

#### Problem
Challenges.md point C1 - Package and Import Layout Is Not Fully Resolved

### Decision

Instead of defaulting to special dir for source files, use project source dirs as source
- assuming we can discover transform by @transform decorators. Generated package structure follow source structure: 
src/my_package becomes generated/structure_generated/my_package. 

Since some python projects use 'src' for source (src/my_package) and other/simpler ones may have packages in project 
root, we need to follow that, e.g. by specifying source roots.

Explicit configuration wins. Without config, discovery uses this convention:

1. If `./src` exists and contains importable packages or modules, use `["src"]`.
2. Otherwise, use `["."]`.

Generated artifact paths mirror module paths relative to the selected source root. The physical root name
itself is not part of the module path.
